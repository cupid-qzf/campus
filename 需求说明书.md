# 校园二手交易平台需求说明书

## 1. 项目概述

### 1.1 项目背景
随着高校学生数量的增加，校园内的二手物品交易需求日益增长。传统的线下交易方式存在信息不对称、交易效率低、安全隐患等问题。为了解决这些问题，开发一个便捷、安全、高效的校园二手交易平台显得尤为必要。

### 1.2 项目目标
- 提供一个方便、快捷的校园二手物品交易平台
- 实现用户注册、登录、商品发布、浏览、搜索、购买等核心功能
- 确保交易过程的安全性和可靠性
- 提供良好的用户体验和界面设计

### 1.3 适用范围
本平台主要面向高校学生和教职工，用于校园内的二手物品交易活动。

## 2. 功能需求

### 2.1 用户管理模块

#### 2.1.1 用户注册
- 支持手机号注册
- 填写基本信息（昵称、密码等）
- 上传头像

#### 2.1.2 用户登录
- 支持手机号和密码登录
- 自动登录功能

#### 2.1.3 个人信息管理
- 修改昵称、密码
- 更新头像
- 查看个人资料

#### 2.1.4 地址管理
- 添加收货地址
- 修改/删除地址
- 设置默认地址

### 2.2 商品管理模块

#### 2.2.1 商品发布
- 填写商品名称、价格、描述
- 选择商品分类
- 上传商品图片
- 设置交易地点

#### 2.2.2 商品浏览
- 首页商品展示
- 分类浏览（数码、家电、户外、图书、其他）
- 分页显示

#### 2.2.3 商品搜索
- 按商品名称搜索
- 按分类筛选

#### 2.2.4 商品详情
- 查看商品详细信息
- 查看卖家信息
- 收藏商品

### 2.3 订单管理模块

#### 2.3.1 下单
- 选择商品
- 确认收货地址
- 提交订单

#### 2.3.2 订单支付
- 模拟支付功能
- 记录支付状态

#### 2.3.3 订单列表
- 查看已购买商品订单
- 查看已卖出商品订单
- 订单状态管理

### 2.4 收藏模块

#### 2.4.1 收藏商品
- 一键收藏商品
- 收藏状态显示

#### 2.4.2 收藏列表
- 查看收藏的商品
- 取消收藏

### 2.5 消息模块

#### 2.5.1 商品留言
- 对商品进行留言咨询
- 查看留言记录

#### 2.5.2 消息回复
- 回复他人留言
- 查看回复内容

### 2.6 后台管理模块

#### 2.6.1 管理员登录
- 管理员账号登录

#### 2.6.2 用户管理
- 查看用户列表
- 禁用/启用用户

#### 2.6.3 商品管理
- 查看商品列表
- 审核商品
- 下架商品

#### 2.6.4 订单管理
- 查看订单列表
- 管理订单状态

## 3. 非功能需求

### 3.1 性能需求
- 页面加载时间不超过3秒
- 系统响应时间不超过2秒
- 支持至少1000并发用户

### 3.2 安全性需求
- 密码加密存储
- 用户认证和授权
- 防止SQL注入、XSS攻击等安全威胁
- 敏感数据传输加密

### 3.3 可用性需求
- 系统可用性达到99%
- 界面友好，易于操作
- 提供错误提示和帮助信息

### 3.4 可扩展性需求
- 系统架构支持横向扩展
- 模块化设计，便于功能扩展

### 3.5 兼容性需求
- 支持主流浏览器（Chrome、Firefox、Safari、Edge）
- 响应式设计，适配不同设备屏幕

## 5. 系统架构

### 5.1 架构模式
采用前后端分离架构：
- 前端：Vue.js + Element UI
- 后端：Spring Boot + MyBatis
- 数据库：MySQL

### 5.2 系统分层
- **表示层**：用户界面，处理用户交互
- **业务逻辑层**：处理核心业务逻辑
- **数据访问层**：与数据库交互
- **数据库层**：存储系统数据
