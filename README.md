# 校园二手交易平台

## 项目概述
校园二手交易平台是一个基于前后端分离架构的在线交易系统，专为校园用户提供便捷的二手物品交易服务。该平台允许用户发布、浏览、搜索二手物品，进行交易沟通，并提供管理员后台管理功能。

## 技术栈

### 后端技术
- **框架**: Spring Boot 2.4.0
- **持久层**: MyBatis + MySQL 8.0.22
- **JSON处理**: FastJson 1.2.61
- **工具库**: Lombok、Hutool 5.7.21
- **开发语言**: Java 8

### 前端技术
- **框架**: Vue 2.6.10
- **路由**: Vue Router 3.0.3
- **UI组件库**: Element UI 2.11.0
- **HTTP请求**: Axios
- **开发语言**: JavaScript

## 部署说明

### 环境要求
- JDK 1.8
- Maven 3.6.1
- MySQL
- Navicat（数据库管理工具）
- Node.js 14

### 部署步骤

1. **导入后端项目**
   - 将 `school-trade-backend` 文件夹导入 IDEA
   - 配置 Maven 和 JDK 环境

2. **配置数据库**
   - 使用 Navicat 连接 MySQL
   - 导入数据库脚本 `DB_School_Trade.sql`
   - 修改 `application.yml` 中的数据库连接配置

3. **配置图片路径**
   - 修改 `application.yml` 中的图片上传路径

4. **启动后端服务**
   - 运行 `BackendApplication` 类启动后端

5. **启动前端服务**
   - 进入 `school-trade-frontend` 目录
   - 执行 `npm install` 安装依赖
   - 执行 `npm run dev` 启动前端

6. **访问系统**
   - 在浏览器中输入前端运行后的地址
   - 使用提供的账号登录系统
